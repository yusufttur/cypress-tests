# Cypress Tests

This repository contains some cypress tests.

## Getting Started

1. Clone the repository:
   ```bash
   git clone https://github.com/yusufttur/cypress-tests.git
   ```
2. Install dependencies:
   ```bash
   npm install
   ```
3. Open cypress runner
   ```bash
   npx cypress open
   ```
4. Run all tests headless
   ```bash
   npx cypress run
   ```
